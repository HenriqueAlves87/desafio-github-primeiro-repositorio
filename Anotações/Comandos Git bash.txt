Alguns comandos do Git Bash

Atalho para limpar a tela:
CTRL + l

Listar arquivos:
ls

Para mudar de pasta:
cd "nome da pasta"

Para mudar para pasta acima:
cd ..

Para ver status da pasta:
git status

Para adicionar arquivos no commit (mudar para staged):
git add .

git add "nome do arquivo com espaços em branco "

Para confirmar todos os arquivos rastreados:
git commit -A

Para "commitar" arquivos no GitHub
git commit -m "mensagem que quiser da commit"

Depois de commit tem dar um push os arquivos para o Git Hub
git push origin main

Comandos:
obs:. no passo 5 depois do nome origin vc vai colar o endereço do seu repositório no github, um bom exemplo logo a baixo.

1 - git init
2 - git branch -m main
3 - git add .
4 - git commit -m "commit final do projeto"
5 - git remote add origin "endereço do repositório no github"
6 - git push -u origin main
7 - Enter
